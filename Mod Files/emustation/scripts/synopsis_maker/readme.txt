This is to be used with gamelist.xml files that are generated by the following applications/tools.

EmulationStation
Scraper.exe ( FastScraper.bat )
Universal XML Scraper

These are all PC tools or programs. EumlationStation is also what powers Retropie and Recalbox.

Usage:
Place your gamelist.xml next to the default.py and then select the default.py script.
Select the emulator the gamelist.xml is for and it will do the rest.



!!!! DO NOT USE THE "gamelist.xml" FILES GENERATED BY XBMC-EMUSTATION. !!!!
